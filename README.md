Resumo
-----
Este repositório contém os artefatos da ênfase **Qualidade de Software** para o projeto SGHSS:
- Plano de Testes (docs/plano_de_testes.md)
- Estratégia de Testes (docs/estrategia_de_testes.md)
- Requisitos priorizados (docs/requisitos.md)
- Casos de Teste (casos_de_teste/casos_de_teste.md)
- Pseudocódigos e exemplos de automação (scripts/)
- Evidências de execução (evidencias/)

Como usar
---------
1. Abra os arquivos em `docs/` e `casos_de_teste/`.
2. Para abrir relatórios JMeter/OWASP ZAP, veja `evidencias/jmeter_reports/` e `evidencias/zap_reports/`.
3. Se quiser executar os exemplos de automação (Cypress), siga as instruções no arquivo `scripts/README.md`.

Estrutura
---------
- docs/: documentos (plano, requisitos, cronograma)
- casos_de_teste/: casos em formato .md ou .xlsx
- scripts/: pseudocódigo e exemplos
- evidencias/: prints e relatórios
